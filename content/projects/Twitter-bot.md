The project aims to detect Twitter bots to prevent the spread of false information using machine learning algorithms. By analyzing both textual and numeric features, the system labels users as genuine or fake. The project compares various algorithms like KNN, SVM, and Adaboost to determine the most effective approach for bot detection.
